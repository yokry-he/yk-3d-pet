# Changelog

All notable changes to this project are documented here.

## 0.1.0 - 2026-07-22

- Extracted the 3D pet authoring Skill from YK-PETS into an independent repository.
- Renamed the Skill to `yk-3d-pet`.
- Added framework-neutral `generic` and YK-PETS integration presets.
- Added portable schemas, prompts, Agent adapters, CLI tools, and smoke tests.
- Added semantic Rig contracts for four limbs, independent eyes, segmented tails, chest/back symbols, props, and effects.
- Added explicit action lifecycle and idle scheduling guidance.
- Added Ubuntu, Windows, and macOS validation.
- Added npm tarball auditing and a guarded tag release workflow.
